**Here's how we can generate a visually appealing graph using `networkx` and `matplotlib` (and a bit of `graphviz` for layout):**

```python
import networkx as nx
import matplotlib.pyplot as plt
import pandas as pd

# Load the data (same as before)
mutation_log = pd.read_csv("mutation_log.txt", delimiter=", ", engine="python")
time_series_data = pd.read_csv("time_series_data.csv")

# Create a directed graph
G = nx.DiGraph()

# Add the Original node
G.add_node("Original", lifespan=(time_series_data['Original'].astype(bool).sum()), peak=time_series_data['Original'].max())

# Function to add mutants and edges to the graph
def add_mutant_to_graph(parent, mutant_id, time, mutation_type, change):
    # Determine lifespan and peak for the mutant
    lifespan_start = time_series_data[mutant_id][time_series_data[mutant_id] > 0].index.min() + 1
    lifespan_end = time_series_data[mutant_id][time_series_data[mutant_id] > 0].index.max() + 1
    peak = time_series_data[mutant_id].max()

    # Add the mutant node with lifespan and peak as attributes
    G.add_node(mutant_id, lifespan=(lifespan_start, lifespan_end), peak=peak)

    # Add an edge from parent to mutant with mutation details as attributes
    G.add_edge(parent, mutant_id, time=time, type=mutation_type, change=change)

# Parse mutation_log.txt and add nodes/edges to the graph
for index, row in mutation_log.iterrows():
    time = int(row['Time'])
    parent = row['Parent']
    mutant_id = f"M{index + 1}"
    mutation_type = row['Type']
    change = row['Change']

    add_mutant_to_graph(parent, mutant_id, time, mutation_type, change)

# Use Graphviz for layout (you may need to install pygraphviz: pip install pygraphviz)
pos = nx.nx_agraph.graphviz_layout(G, prog="dot")

# Draw the graph
plt.figure(figsize=(18, 12))

# Node labels with lifespan and peak
node_labels = {node: f"{node}\nLife: {data['lifespan']}\nPeak: {data['peak']}" for node, data in G.nodes(data=True)}
nx.draw_networkx_labels(G, pos, labels=node_labels, font_size=8)

# Edge labels with mutation details
edge_labels = {(u, v): f"T:{d['time']}, {d['type']}: {d['change']}" for u, v, d in G.edges(data=True)}
nx.draw_networkx_edge_labels(G, pos, edge_labels=edge_labels, font_size=6, label_pos=0.3, rotate=False)

# Draw nodes (scaled by peak infection)
nx.draw_networkx_nodes(G, pos, node_size=[data['peak'] / 5 for node, data in G.nodes(data=True)], node_color="skyblue")

# Draw edges
nx.draw_networkx_edges(G, pos, width=1, edge_color="gray", arrows=True, connectionstyle="arc3,rad=0.1")

plt.title("Memetic Family Tree of the Recursive Infinity Meme")
plt.axis("off")
plt.show()

```

**Explanation of Changes:**

2. **Create a Directed Graph:** `G = nx.DiGraph()` creates a directed graph object to represent the tree.
3. **Add Nodes and Edges:** The `add_mutant_to_graph` function now adds nodes and edges to the `G` object, including attributes like `lifespan` and `peak` for nodes, and `time`, `type`, and `change` for edges.
4. **Graphviz Layout:**
    *   `pos = nx.nx_agraph.graphviz_layout(G, prog="dot")` uses the `dot` layout engine from Graphviz to calculate node positions. This generally produces a nice tree-like structure. You might need to install `pygraphviz` (e.g., `pip install pygraphviz`).
5. **Drawing:**
    *   `plt.figure(figsize=(12, 8))` sets the figure size.
    *   `nx.draw_networkx_labels` draws node labels with lifespan and peak information.
    *   `nx.draw_networkx_edge_labels` draws edge labels with mutation details.
    *   `nx.draw_networkx_nodes` draws the nodes, scaling their size by the `peak` value.
    *   `nx.draw_networkx_edges` draws the edges with arrows.
6. **Title and Axis:** `plt.title` adds a title, and `plt.axis("off")` hides the axis.
